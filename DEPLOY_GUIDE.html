🚀 Firebase Deployment Guide
ℹ This guide walks you through deploying your Excellent Susu React app to Firebase Hosting — a free, fast, and secure hosting service by Google.
After deployment your app will be live at a URL like:
https://excellent-susu.web.app
or https://excellent-susu.firebaseapp.com
You can also connect a custom domain (e.g. excellentsusu.com) for free in the Firebase console.

✓ Prerequisites

Make sure you have these installed on your computer before starting:
  • Node.js (v18+) — download from nodejs.org
  • npm — comes with Node.js automatically
  • A Google account — to access Firebase console
  • A terminal / command prompt — Terminal on Mac, CMD or PowerShell on Windows
ℹ Check your Node version by running node --version. It should show v18 or higher.

1 Create a Firebase Project

Go to the Firebase console and create a new project:
  1. Open your browser and go to console.firebase.google.com
  2. Click "Add project"
  3. Enter project name: excellent-susu
  4. Disable Google Analytics (not needed) → Click "Create project"
  5. Wait for it to finish → Click "Continue"
  6. In the left sidebar, click Hosting → Click "Get started"

2 Set Up Your Project Folder

You received a ZIP file with the complete project. Here is what's inside:
excellent-susu/
  ├── public/
    │   └── index.html
  ├── src/
    │   ├── index.js
    │   └── App.jsx  ← Your full app
  ├── package.json
  ├── firebase.json  ← Firebase config
  ├── .firebaserc   ← Links to your project
  └── .gitignore
⚠ Open the .firebaserc file and replace "excellent-susu" with your actual Firebase Project ID if it's different.

3 Install Firebase CLI

Open your terminal and run this command to install the Firebase tools globally:
npm install -g firebase-tools
Then log in to your Google account:
firebase login
ℹ A browser window will open — sign in with the Google account you used for Firebase. Come back to the terminal after.

4 Install App Dependencies

Navigate into your project folder and install React dependencies:
cd excellent-susu
npm install
This will download all required packages into a node_modules/ folder. It may take 1–2 minutes.

5 Build the App

Create a production-optimised build of your app:
npm run build
This creates a build/ folder with compressed, optimised files ready for deployment. You should see a success message like Compiled successfully.
✓ If you see warnings, that's fine. Only errors (in red) need to be fixed before continuing.

6 Deploy to Firebase

You're ready! Deploy with a single command:
firebase deploy
After a few seconds you'll see output like:
✔  Deploy complete!

Project Console: https://console.firebase.google.com/project/excellent-susu
Hosting URL:     https://excellent-susu.web.app
🎉 Your app is now live! Open the Hosting URL in any browser.

↻ Updating the App Later

Every time you make changes to your app, just run these two commands again from the project folder:
npm run build
firebase deploy

🌐 Connect a Custom Domain (Optional)

To use your own domain like excellentsusu.com:
  1. Go to Firebase Console → Hosting → Add custom domain
  2. Enter your domain name
  3. Firebase will give you DNS records — add them in your domain registrar (GoDaddy, Namecheap, etc.)
  4. Wait up to 24 hours for DNS to propagate
  5. Firebase will automatically provision an SSL certificate (HTTPS) for free ✓

⚠ Common Issues

❌ "firebase: command not found"
Run npm install -g firebase-tools again. On Mac you may need to prefix with sudo.
❌ Build fails with errors
Read the red error message carefully. Most common fix: make sure you ran npm install first.
❌ Wrong project deployed
Open .firebaserc and confirm the project ID matches exactly what's in your Firebase console.
❌ App shows blank page
Open browser DevTools (F12) → Console tab and check for errors. Most often a missing file or import issue.
🚀 Quick Deploy Checklist
  • Firebase project created at console.firebase.google.com
  • Hosting enabled in the Firebase console
  • Project ID in .firebaserc matches your Firebase project
  • npm install -g firebase-tools done
  • firebase login done
  • npm install inside project folder done
  • npm run build completed successfully
  • firebase deploy — app is live! 🎉
